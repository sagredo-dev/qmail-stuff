NAME

       ezmlm-unsub  -  manually  remove  addresses from a mailing
       list


SYNOPSIS

       ezmlm-unsub dir [ box@domain ...  ]


DESCRIPTION

       ezmlm-unsub removes each address box@domain from the mail-
       ing list stored in dir.

       If  box@domain  is  not  on  the mailing list, ezmlm-unsub
       keeps it off.

       ezmlm-unsub converts domain to lowercase  before  removing
       box@domain from the mailing list.
