NAME

       ezmlm-sub - manually add addresses to a mailing list


SYNOPSIS

       ezmlm-sub dir [ box@domain ...  ]


DESCRIPTION

       ezmlm-sub adds each address box@domain to the mailing list
       stored in dir.

       If box@domain is already on the  mailing  list,  ezmlm-sub
       leaves it there.

       ezmlm-sub  converts  domain  to  lowercase  before  adding
       box@domain to the mailing list.

       box@domain cannot be longer than 400 characters.
