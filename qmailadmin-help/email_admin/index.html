Email Administration Guide

QmailAdmin is a web based tool that allows you to create, edit, and delete mailboxes for your domain. You can also create mail lists, forwards and aliases. Users can set vacation messages or forward email. Some sites allow domain administrators to set quotas.

Most sites qmailadmin can be accessed by going to https://mail.server/cgi-bin/qmailadmin
